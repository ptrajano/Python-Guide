## Exercícios Iniciantes

**Problema:** Crie uma função que recebe uma lista de números e retorne:
- A soma de todos os elementos
- O maior número da lista
- A lista em ordem crescente


**Problema:** Crie uma função que recebe um dicionário de produtos e preços, e retorne:
- O produto mais caro
- A média dos preços
- Um novo dicionário com produtos que custam mais de R$ 50,00


**Problema:** Crie uma função que recebe uma tupla de números e retorne:
- Uma lista com os números pares
- Uma lista com os números ímpares
- A quantidade de elementos


**Problema:** Crie uma função que recebe uma lista com diferentes tipos de dados (números, strings, booleanos) e retorne:
- Quantos números existem na lista
- Quantas strings existem na lista
- Quantos booleanos existem na lista


**Problema:** Crie uma função que recebe uma lista de strings e retorne:
- Uma nova lista com todas as strings em maiúsculo
- A string mais longa
- A string mais curta

## Exercícios Avançados

**Problema:** Crie uma função que receba um
texto do usuário e salve ele em um arquivo.

**Problema:** Crie uma função que divide dois números e trata o erro de divisão por zero.

**Problema:** Peça um número ao usuário e trate erros de conversão.

**Problema:** Crie uma função que peça números para o usuário. A função deve:
1. Pedir para o usuário digitar um número
2. Se digitar um número válido, guardar em uma lista
3. Se digitar texto que não é número, mostrar mensagem de erro e pedir novamente
4. Se digitar a palavra "exit", parar de pedir números
5. No final, retornar a lista com todos os números que o usuário digitou

**Exemplo de como deve funcionar:**

```text
Digite um número (ou 'exit' para sair): 10
Digite um número (ou 'exit' para sair): 20
Digite um número (ou 'exit' para sair): abc
Erro: Por favor, digite apenas números!
Digite um número (ou 'exit' para sair): 30
Digite um número (ou 'exit' para sair): exit
```


A tupla é basicamente uma [[Lista]], com a mudança que esse objeto é imutável. Para gerar uma tupla existe duas formas, a primeira é usar parenteses

```python
my_tuple = (1, 2, 3)

print(my_tuple)
```

A outra é usando a função [[Funções Nativas#tuple|tuple]].

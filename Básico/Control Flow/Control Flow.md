A forma básica que uma linguagem de programação funciona é que cada linha é executada de forma linear, a primeira linha é executada então a segunda e assim sucessivamente, as estruturas de controle de fluxo modificam o fluxo padrão, podendo pular linhas [[If, Elif e Else]] e [[Match case]] ou repetir as mesmas linhas várias vezes [[For]] e [[While]]. Essas estruturas são essenciais para códigos mais complexos, pois dão a liberdade para comportamentos não lineares.


Essa lista é pessoal, mas foi usado [open source libs](https://www.mygreatlearning.com/blog/open-source-python-libraries/) e [python libs](https://opencv.org/blog/top-python-libraries/), para falar das bibliotecas mais importantes de se conhecer no `python`
# Ciência de Dados

## [Pandas]([https://pandas.pydata.org/docs/](https://pandas.pydata.org/docs/))

Biblioteca fundamental para manipulação e análise de dados que fornece estruturas de dados de alto desempenho e fáceis de usar, como DataFrames e Series, permitindo operações eficientes de limpeza, transformação, agregação e visualização de dados tabulares.

## [Matplotlib]([https://matplotlib.org/stable/contents.html](https://matplotlib.org/stable/contents.html))

Biblioteca completa e amplamente adotada para criação de visualizações estáticas, animadas e interativas em Python, oferecendo controle granular sobre todos os elementos de um gráfico através de sua API orientada a objetos e interface estilo MATLAB.

## [Seaborn]([https://seaborn.pydata.org/](https://seaborn.pydata.org/))

Biblioteca de visualização baseada no Matplotlib que fornece uma interface de alto nível para criar visualizações estatísticas atraentes e informativas, com estilos padrão atraentes e ferramentas especializadas para visualizar dados complexos.

# Inteligência Artificial

## [Scikit-learn]( [https://scikit-learn.org/stable/documentation.html](https://scikit-learn.org/stable/documentation.html))

Biblioteca de machine learning que oferece algoritmos simples e eficientes para análise preditiva, incluindo classificações, regressões, clustering e redução de dimensionalidade, com interface consistente e documentação extensa.

## [Keras]([https://keras.io/](https://keras.io/))

API de rede neural de alto nível que funciona como interface para frameworks como TensorFlow, oferecendo abstrações intuitivas para construir e treinar modelos de deep learning com foco na experiência do desenvolvedor e rápido prototipagem.

## [PyTorch]([https://pytorch.org/docs/stable/index.html](https://pytorch.org/docs/stable/index.html))

Framework de deep learning que oferece computação tensorial com aceleração GPU e redes neurais profundas baseadas em grafo de computação dinâmico, sendo amplamente adotado na pesquisa acadêmica e aplicações de produção.

# Web
## [Flask](https://flask.palletsprojects.com/)

Microframework web minimalista e flexível que fornece o essencial para construir aplicações web, permitindo ao desenvolvedor tomar decisões sobre ferramentas e bibliotecas adicionais, ideal para APIs e aplicações simples.

## [Django](https://docs.djangoproject.com/)

Framework web full-stack de alto nível que segue o padrão MVC e inclui ORM próprio, sistema de autenticação, administração automática e diversas outras funcionalidades built-in para desenvolvimento rápido de aplicações complexas.

## [Requests](https://docs.python-requests.org/)  

Biblioteca elegante e simples para fazer requisições HTTP que abstrai a complexidade das bibliotecas padrão, oferecendo interface intuitiva para métodos GET, POST e outros com suporte a autenticação, cookies e sessões.

## [BeautifulSoup](https://www.crummy.com/software/BeautifulSoup/bs4/doc/)  

Biblioteca para parsing de documentos HTML e XML que fornece maneiras idiomáticas de navegar, pesquisar e modificar árvores de parse, sendo especialmente útil para web scraping e extração de dados de páginas web.

## [Selenium](https://www.selenium.dev/documentation/)  

Ferramenta de automação web que permite controlar navegadores programaticamente para testar aplicações web, automatizar tarefas e fazer scraping de conteúdo gerado dinamicamente por JavaScript.

## [FastAPI](https://fastapi.tiangolo.com/)  

Framework web moderno de alto desempenho para construir APIs com Python baseado em type hints, com geração automática de documentação OpenAPI e validação de dados através do Pydantic.

# Cientifica
## [NumPy](https://numpy.org/doc/)  

Biblioteca fundamental para computação científica que fornece objetos de array multidimensionais e funções matemáticas para operações eficientes com grandes volumes de dados, servindo como base para todo o ecossistema científico.

## [SciPy](https://docs.scipy.org/doc/scipy/)  

Coleção de algoritmos matemáticos e funções convenientes built-on no NumPy para tarefas como integração numérica, otimização, álgebra linear, processamento de sinais e estatística avançada.

## [SymPy](https://docs.sympy.org/latest/index.html)

Biblioteca para matemática simbólica que permite realizar álgebra, cálculo, equações diferenciais e outras operações matemáticas de forma exata usando símbolos em vez de aproximações numéricas.

# Banco de Dados

## [SQLAlchemy]([https://docs.sqlalchemy.org/](https://docs.sqlalchemy.org/))

Toolkit SQL e Object Relational Mapper que fornece conjunto completo de patterns para acesso a bancos de dados, permitindo trabalhar com banco de dados usando tanto SQL raw quanto ORM de alto nível.

## [pySpark]([https://spark.apache.org/docs/latest/api/python/index.html](https://spark.apache.org/docs/latest/api/python/index.html))

Interface Python para Apache Spark, framework de computação distribuída que permite processar grandes volumes de dados de forma paralela e distribuída com APIs para SQL, streaming e machine learning.

# Miscelâneas 

## [PyGame]([https://www.pygame.org/docs/](https://www.pygame.org/docs/))

Conjunto de módulos Python para desenvolvimento de jogos e aplicações multimídia que fornece funcionalidades para gráficos, som, input e outras necessidades comuns em desenvolvimento de jogos.

## [Pillow]([https://pillow.readthedocs.io/](https://pillow.readthedocs.io/))

Fork da biblioteca PIL que adiciona suporte para abrir, manipular e salvar muitos formatos de imagem diferentes, oferecendo operações como redimensionamento, rotação, filtros e conversão entre formatos.

## [Streamlit]([https://docs.streamlit.io/](https://docs.streamlit.io/))

Framework para criar aplicações web interativas para machine learning e ciência de dados com mínimo de código, transformando scripts Python em aplicações compartilháveis com widgets interativos.

## [OpenCV](https://docs.opencv.org/4.x/](https://docs.opencv.org/4.x/)|)

Biblioteca de visão computacional e aprendizado de máquina que fornece ferramentas abrangentes para processamento de imagem e vídeo, incluindo detecção de objetos, reconhecimento facial e análise de movimento.

Vivemos em uma era extremamente tecnológica, [estima-se](https://www.worldometers.info/pt/computadores/#:~:text=Quantos%20computadores%20existem%20no%20mundo,bilh%C3%B5es%20no%20ano%20de%202015.) que existem mais de dois bilhões de computadores pessoais no mundo, mais de [3 milhões](https://www.radiorock.com.br/noticias/videogames-mundo-tem-mais-de-35-bilhoes-de-jogadores) de jogadores, [serviços de cloud](https://www.google.com/search?q=cloud+services&ie=UTF-8), como a [AWS](https://aws.amazon.com/pt/) tem mais de [4 milhões de usuários](https://hginsights.com/blog/aws-market-report-buyer-landscape#:~:text=Insights%20de%20mercado%20da%20AWS,varejo%2C%20internet%2C%20manufatura%20e%20educa%C3%A7%C3%A3o), a quantidade de sistemas tecnológicos que dependem de programação, como [IoT](https://pt.wikipedia.org/wiki/Internet_das_coisas), carros, máquinas de lavar, celulares, câmeras de segurança, indústrias provavelmente ultrapassam os 10 bilhões. 
Dessa forma compreender a tecnologia deixou de ser apenas um diferencial e passou a ser uma necessidade, por exemplo o excel e o word que são ferramentas básicas para viver em sociedade.
Para uma máquina fazer o que você quer é preciso se comunicar com ela e, nós nos comunicamos com ela através de linguagens de programação, que é a forma que traduzimos nossas ideias para instruções que os computadores ou [sistemas embarcados](https://materialpublic.imd.ufrn.br/curso/disciplina/1/17/10/2#:~:text=Mas%20Afinal%2C%20o%20que%20s%C3%A3o,que%20os%20tornam%20%E2%80%9Csmartphones%E2%80%9D.) consigam entender e executar.
Programar é, portanto, conversar com as máquinas expressando nossas ideias através da lógica. O processo de programar, mesmo que não pareça, é um processo extremamente criativo onde precisamos traduzir conceitos que são naturais para a gente para uma máquina. Por exemplo, como ensinaríamos um robô a andar, parece trivial né? Mas não é, toda a ideia de movimentar uma perna, mesmo que seja fácil para a gente, é um processo complexo de mandar três motores funcionar, o do pé, o do joelho e o da bunda, mas não basta eles funcionarem é preciso que eles funcionem por um tempo específico para que o robô não caia. 
A criatividade entra exatamente nisso, traduzir o que queremos no que a máquina entende e de como vamos escrever o que queremos, afinal, existem diversas maneiras de executar uma mesma tarefa.

## Por quê existem tantas linguagens de programação

Assim como os idiomas humanos, existem diversas formas de se comunicar da máquina, cada uma delas com as suas vantagens e desvantagens, mas por debaixo do pano, as linguagens de programação traduzem o nosso código em [linguagem de máquina](https://www.lenovo.com/br/pt/glossary/machine-language/) que é uma sequência de [números binários](https://www.google.com/search?q=numeros+binarios&ie=UTF-8), mas é quase impossível (e sinceramente masoquista) tentar programar diretamente nessa linguagem, pois ela não é "humana". Para "humanizar" o processo de programar linguagens de programações foram criadas, cada uma com um grau de "humanização" da linguagem. 
Um bom programador sabe sobre lógica de programação, o que que isso quer dizer? Um bom programador não é alguém que sabe apenas uma linguagem, é essencial que saiba-se várias, pois, como foi dito anteriormente, cada linguagem tem suas vantagens e desvantagens. Além disso um programador não pode ficar preso no tempo, se especializando em somente uma linguagem, que provavelmente, alternativas melhores apareçam mais para frente, entender lógica de programação facilita no processo de entendimento de novas linguagens.
Dito isso, esse curso será focado somente na linguagem de programação `Python`, apresentando um pouco sobre lógica de programação, mas seu foco é a linguagem em si.

## O que é Python

O [Python](https://www.python.org/) é uma linguagem de [programação interpretada](google.com/search?q=interpreter+vs+compiled&oq=interpreter+vs+compiled&gs_lcrp=EgZjaHJvbWUyBggAEEUYOTIGCAEQLhhA0gEINjczNGowajGoAgCwAgA&sourceid=chrome&ie=UTF-8) (programa executa linha por linha separadamente), de [alto nível](https://en.wikipedia.org/wiki/High-level_programming_language) (mais "humanizada"), e multiplataforma (roda em Windows, Linux, Ubuntu, etc...). Ela foi criada por [Guido Von Rossum](https://pt.wikipedia.org/wiki/Guido_van_Rossum) no início dos anos 1990.
A sintaxe do `Python` é simples e legível, o que a torna ideal tanto para programadores iniciantes quanto para profissionais experientes.
Com o passar dos anos, o  `Python` se consolidou como uma das linguagens mais utilizadas no mundo, e a partir de 2023, se tornou a [linguagem com mais programadores](https://www.tiobe.com/tiobe-index/), onde $24\%$ dos programadores sabem escrever nela, além disso com os [dados da jetbrains](https://lp.jetbrains.com/python-developers-survey-2022/), vemos que, em 2022, o `python` é a linguagem principal de $85\%$ dos desenvolvedores, além disso `python` é usado em diversas áreas, como analise de dados, desenvolvimento web, aprendizado de máquina, automação e acadêmica.

## Por que aprender Python

Acredito que partindo do que falamos é claro o motivo de aprender `python`, para os iniciantes, é uma linguagem fácil de entender e programar, com sintaxe fácil e intuitiva, além do que por ser uma das maiores linguagens de programação, eu chuto dizer, que é a linguagem mais documentada e com mais códigos na internet, isso pode ser visto usando LLM's como o chatgpt, que ao pedir para ele fazer um código, sem especificar a linguagem, o código que ele dará será em `python`. Para programadores mais experientes, `python` é uma linguagem fácil para fazer projetos simples, além de seu mercado de trabalho ser gigante e em diversos setores, podendo trabalhar com inteligência artificial, desenvolvimento web ou analise de dados.
Além disso `python` é uma linguagem de programação que sempre esta evoluindo e que, desde sempre foi uma linguagem da comunidade para a comunidade, ou seja, o feedback dos desenvolvedores de fato modificam o que é implementado nela.

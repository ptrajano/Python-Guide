**Mini-Curso Python:

Este mini curso é pensado para aperfeiçoar programadores que já possuem familiaridade com programação e desejam se aprofundar no sistema Python. O intuito do curso é ir além da sintaxe básica e permear conceitos que são um diferencial para o mercado da programação e fazem de Python tão abrangente.

*Dia 1 & 2: Python Básico*  
Revisão dos fundamentos com ênfase em particularidades do Python

- Input/Output e manipulação de dados
    
- Tipos de dados e estruturas nativas do Python
    
- Controle de fluxo e compreensões de lista
    
- Funções: args, kwargs, escopo, funções e lambda
    

*Dia 3 & 4: Python Avançado*  
Aspectos que diferenciam programadores Python iniciantes de experientes

- Decorators: ampliando funcionalidades
    
- Gestão de exceções e context managers
    
- Programação orientada a objetos no estilo Python
    
- Iteradores, geradores e expressões geradoras
    

**Dia 5: Além da linguagem**  
_ Por quê que entender a linguagem não faz de você um bom programador? _

- Estruturas de dados avançadas e suas aplicações
    
- Padrões de design aplicados em Python
    
- Arquitetura e desenvolvimento de projetos escaláveis
    
- Convenções, ferramentas e práticas da comunidade Python
    

_Pré-requisito: Conhecimento básico de programação em qualquer linguagem._
